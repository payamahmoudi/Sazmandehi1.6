# سامانه ساماندهی نیروی انسانی آموزش و پرورش

## نسخه تحت وب

### نحوه استقرار
1. پروژه را Build کنید: `npm run build`
2. محتوای پوشه `dist` را روی وب‌سرور (Nginx, Apache, ...) قرار دهید
3. سامانه از طریق آدرس وب‌سایت قابل دسترسی خواهد بود

### امنیت
- تمام رمزهای عبور Hash شده ذخیره می‌شوند
- نشست کاربر با بستن مرورگر حذف می‌شود
- خروج خودکار بعد از ۳۰ دقیقه عدم فعالیت
- فایل‌های پشتیبان رمزنگاری شده هستند
- کلیک‌راست و ابزارهای توسعه‌دهنده غیرفعال هستند
- انتخاب متن و Drag غیرفعال است
- هر کاربر فقط به داده‌های خودش دسترسی دارد
- مدیر اصلی به تمام پشتیبان‌ها دسترسی دارد

### پشتیبان‌گیری
- از بخش `ورود/خروج` → `پشتیبان‌گیری رمزنگاری‌شده`
- فایل پشتیبان فقط با حساب کاربری صاحب آن قابل بازیابی است
